# Brane Hub 🚀

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![GitHub Stars](https://img.shields.io/github/stars/deepsuthar496/Brane.svg)](https://github.com/deepsuthar496/Brane/stargazers)

**Brane Hub** is a powerful, centralized control plane and GUI designed exclusively for top-tier AI coding agents: **Gemini CLI**, **Claude Code**, and **Codex**. 

Stop juggling terminal windows, manually editing `.env` files, or struggling with complicated MCP server configurations. Brane Hub automatically discovers your installed agents and provides a beautiful, unified dashboard to manage them, configure them, and turbocharge their capabilities.

---

## 🔥 Core Features

- **🔌 Exclusive Agent Discovery**: Automatically detects and integrates exclusively with `gemini`, `claude`, and `codex`.
- **🚩 Interactive Flags Manager**: Real-time discovery and configuration of CLI flags. Toggle flags like `--yolo` or `--model` visually with one click.
- **🧩 MCP Central**: The easiest way to manage Model Context Protocol (MCP) servers. Enable or disable servers and sync them seamlessly across your supported agents.
- **⚡ Skill Registry**: Extend agent capabilities by downloading specialized instructions and tools (Skills) from a curated remote registry directly to your agents.
- **🛡️ Secure Credentials**: A unified, secure vault for your API keys. No more scattering secrets across different `.env` files.
- **🧠 Knowledge Base**: Inject your local project context, architectural diagrams, and documentation directly into your agents' context window.

## 🚀 Quick Start

### For Users (Windows)
1. Go to the [Releases](https://github.com/deepsuthar496/Brane/releases) page.
2. Download the latest **`BraneHub_Setup.exe`** installer.
3. Run the installer and launch Brane Hub.

### For Developers (Run from Source)
```bash
# Clone the repository
git clone https://github.com/deepsuthar496/Brane.git && cd Brane

# Install dependencies
npm install

# Launch in dev mode
npm run electron:dev
```

## 🛠️ Tech Stack
- **Frontend**: Next.js (Turbopack), Tailwind CSS, Shadcn UI
- **Backend**: Electron (Node.js)
- **AI Integration**: AI SDK (Vercel), Model Context Protocol (MCP)
